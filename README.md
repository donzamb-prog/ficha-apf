# Autorização do APF - GE Memorial 350/SP

Este é um sistema web responsivo desenvolvido especialmente para o **Grupo Escoteiro Memorial 350/SP** para o preenchimento, assinatura digital e geração de PDF da **Autorização do Assessor Pessoal de Formação (APF)** para participação em Cursos Técnicos.

## 🚀 Funcionalidades

- **Preenchimento Simples:** Formulário totalmente estruturado e adaptado para celulares e computadores.
- **Assinatura Digital Integrada:** Permite que o APF e o Diretor Presidente assinem diretamente na tela do celular ou com o mouse.
- **Geração de PDF em 1 Página:** Algoritmo otimizado para alinhar os dados perfeitamente e garantir que todo o documento caiba estritamente em uma única página A4.
- **Compartilhamento Direto:** Botão para compartilhar o arquivo PDF real diretamente via WhatsApp, e-mail ou outros aplicativos do celular utilizando a tecnologia nativa do aparelho.
